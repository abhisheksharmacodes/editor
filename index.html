z - scaling

It clearly doesn't seem any change while you are scaling z the box but it helps to improve 3D view when rotating.
box positioning

positioning of box in editor is too easy you just need to click and hold on box and move whether you want to drop it.
perspective

when you look a building anywhere then it makes a 3D angle with itself which makes it real , as it is editor supports 3D perspection of box. perspective is a property defined in transform tab through which you can control the 3D view whenever you rotate the box. editor also allows you to decide perspection origin. As default 3D view in editor is turned off for ease of normal purpose box designing. you can turn on through the settings.
double texting

double texting means write a letter once a time but box displays it twice. you need to fix some text formatting as follows :
step 1 : open 'text' tab .
step 2 : set more than zero value into second input box of shadow property.
step 3 : set '0' value in third input box.

note that shadow color is the color of copied text.
states
there are three states of box in editor : normal, hover and active.
  • normal state is when you aren't mouseover or clicking the box.
  • hover state is when you mouseover on the box. if you select hover option and make changes to box then it apply only when you're hovering it.
  • active state is when you are clicking the box. if you choose active option and make changes to box then it apply only when you're clicking the box.
